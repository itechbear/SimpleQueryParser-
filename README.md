# SimpleQueryParser++
A simple thread-safe query parser generated by flex and bison

Introduction
============

This is a simple query parser generated by flex and bison that recognizes logic queries like:

 - this AND that
 - this OR that
 - this AND NOT that
 - NOT (this AND that)

Compile
=======

 - $ mkdir build
 - $ cd build
 - $ cmake ..
 - $ make
